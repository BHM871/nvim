## Configurations

### Linux and MacOS

<p>Move the file <a href="https://github.com/BHM871/nvim/init.lua">init.lua</a> and folder <a href="https://github.com/BHM871/nvim/lua">lua</a> to directory:</p>

<code>~/.config/nvim</code>

### Windows

<p>Move the file <a href="https://github.com/BHM871/nvim/init.lua">init.lua</a> and folder <a href="https://github.com/BHM871/nvim/lua">lua</a> to directory:</p>

<code>~/AppData/Local/nvim</code>
